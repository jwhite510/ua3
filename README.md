# Urban Assault 3
A remake inspired by Urban Assault (1998) for PC, made in Unity Engine

# Gameplay
Capture Tiles to increase your resource regeneration rate  
Spawn new Units  
All motion is simulated with rigid body physics, springs, hinges, wheels etc
